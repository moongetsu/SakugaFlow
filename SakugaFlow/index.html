About

SakugaFlow v0.1b

SakugaFlow is a free unofficial After Effects CEP extension made for reference and editing workflows.

It searches Sakugabooru, previews clips, downloads selected files locally, and imports them into After Effects.

Project

SakugaFlow is an independent project. It shares no code with Sakugaloader and was built from scratch with a different structure and features.

Community
Credits & Background

Credit for the original concept of a dedicated Sakugabooru fetcher extension for After Effects goes to the creators of "Sakugaloader".

Why this project was created

When the concept of an After Effects Sakugabooru loader was first released, I wanted to contribute to its development. Specifically, I wanted to help fix bugs, optimize the code, and maintain support for older versions of After Effects.

Because collaboration was not possible and support for older AE versions was dropped in the original tool, I decided to build a new alternative from scratch.

The goal is to ensure editors have access to a lightweight, stable loader regardless of their hardware or software setup. This extension is written entirely from the ground up, shares no code with other plugins, and supports CC 2018 through CC 2026.

Purpose

This project is an independent effort to offer the AMV community a simple, high-performance alternative. Building alternative tools is a standard practice in software development, and I respect the concept that inspired this project.

All code was written from scratch without using any resources from the original plugin. It is open-source, community-focused, and designed to give editors another choice for their workflow.

Made for the AMV community.
